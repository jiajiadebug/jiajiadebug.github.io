Tools that can help you test and debug your regular expressions (正则表达式的调试工具)

  • Tools

Regular Expressions

Today Regular Expressions (regexes) are widely supported in programming languages, text processing programs, advanced text editors, and some other programs. Regex support is part of the standard library of many programming languages, including Java and Python, and is built into the syntax of others, including Perl and ECMAScript.

Texts or strings that need to be carefully deal with are very common in our daily work.

Debug Tools

I would like to share two useful tools that can help you test and debug your regular expressions.

  • https://regex101.com/

Online regex tester for PHP, PCRE, JavaScript and Python that highlights pattern and matches on the fly.

  • http://www.regexlab.com/mtracer

A visual assistant to help you to write, test and debug your regex, it is especially good at long and complex regex. Regex Match Tracer makes your regex easy and clear and efficient. Current free edition without limit is v2.1.5.

中文简单概括

正则表达式是程序员必要掌握的技能。在平时的工作中，很多工具都支持并采用了正则表达式。如何快速有效地调试自己写出的表达式？在本篇文章中，我推荐两个有用的调试工具。

  • https://regex101.com/

一个在线的网站，可以用于调试PHP, PCRE, JavaScript和Python中的正则表达式。支持高亮格式和实时匹配。

  • http://www.regexlab.com/mtracer

一个本地工具。可以用于调试复杂的正则表达式。当前免费版本为v2.1.5
