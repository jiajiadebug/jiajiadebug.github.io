选择喜欢的博客主题

为了快速搭建博客，我们可以直接使用已有的博客主题(theme)，然后慢慢将其个性化（修修补补一天又一天，哈哈）。留点时间和力气吃肉嘛。

  • 在已有的博客主题中选择一个喜欢的，在github中找到它，fork一下。然后clone到themes文件夹。
  • 启用这个主题。在配置文件_config.yml中找到theme: XXXX，修改成你选择的主题。我这里是yelee。（注意：Hexo有两个_config.yml文件，一个在根目录，一个在theme下，此时修改的是在根目录下的。）
  • 重启Server，刷新一下浏览器。现在这个博客上采用的就是刚刚你选择的主题。

啦啦啦，现在可以修改主题，将其个性化。比如改图片，放点萌照啦。。。压抑住情绪，继续部署。。。

注册域名绑定博客

  • 购买域名

我是通过aliyun购买的域名，购买过程挺简单。首先查询一下想注册的域名是否被注册，然后加入订单，选择年数，支付宝付款即可。

  • 解析

添加如下规则到解析列表中。启动它。然后等待。（注意io后面有一个.）

CNAME @ username.github.io.

  • 状态

吐槽一下，没有实名，购买五天后，被hold住（暂停解析了），没有任何通知。T T。域名实名一提交，虽然还在待审核，但是域名马上可以用了。建议装阿里云到手机，可以及时知道域名状态。

更新主题和博文

  • 更新主题

推荐先enable那些自己觉得最不可或缺的功能或模块。小细节的东西可以在以后慢慢更新。

  • 更新博文

可以在命令中指定文章的布局（layout），默认为post，可以通过修改_config.yml中的default_layout参数来指定默认布局。Hexo 有三种默认布局：post、page 和 draft，它们分别对应不同的路径(source/_posts, source, source/_drafts)。

  • 推荐Markdown工具

MacDown不错。清爽，基本功能都有了。下载即可使用。
